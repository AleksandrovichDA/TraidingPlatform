Разрабатываем приложение систему помощника для трейдера.  

Данная система представляет собой портал который позволяет просматривать и контролировать открытые позиции. Вести дневник со скриншотами. Торговать с разными стратегиями и собирать статистику таких стратегий (точку входа в позицию, точку выхода). В будущем планируется подключить api для торговли на bitget. Но первая основная функция это робот для контроля управлением рисками при совершении сделки.
